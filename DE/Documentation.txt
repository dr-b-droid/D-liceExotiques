PRESENTATION
================================================================================

Délices Exotiques est une application web full-stack moderne simulant la
gestion complète d'un restaurant gastronomique spécialisé dans la cuisine
africaine et internationale. Ce projet académique combine élégance visuelle,
fonctionnalités avancées et architecture robuste pour offrir une expérience
utilisateur exceptionnelle.

POINTS CLES :
- Interface utilisateur élégante et intuitive
- Système d'authentification sécurisé
- Dashboard administrateur complet
- Gestion en temps réel des commandes
- Systeme de notation dynamique et public du site
- Catalogue de 57+ menus extensible

ACCÈS ADMINISTRATEUR
================================================================================

Le système administrateur est sécurisé et nécessite une authentification en
deux étapes.

ROUTE D'ACCÈS
-------------
http://localhost:3000/admin

PREREQUIS
---------
1. Etre connecté en tant qu'utilisateur
   -> Si non connecté : redirection automatique vers /auth.html

2. Entrer le code administrateur
   Code par défaut : ADMIN2025
   -> Accès via /verifyAdmin.html puis redirection automatique

SESSION ADMINISTRATEUR
----------------------
- Durée : 7 heures
- Sécurité : Vérification IP + Token cryptographique
- Expiration : Redirection automatique + notification

FONCTIONNALITES ADMIN
----------------------
- Dashboard avec statistiques temps réel
- Gestion complète des commandes
- Validation des réservations
- Traitement des événements
- Gestion complète des menus
- Gestion complète des utilisateurs
